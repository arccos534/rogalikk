# rogalikk

Simple memory game made with Python and Tkinter.

The player sees a sequence of correct doors, memorizes it, and then repeats the sequence round by round. A wrong answer ends the current run.
Each new level increases the pressure with longer sequences, more doors, less time to memorize, and higher score stakes.

## Run

1. Install Python 3.
2. Open the project folder.
3. Start the game with `python игра.py`.
